# 🐪 캐멀 러시 3D — 프로젝트 프롬프트

## 프로젝트 개요
보드게임 "Camel Up"을 Three.js 3D 미니게임으로 재구현. 모바일/웹에서 실행되는 단일 HTML 파일.

## 현재 상태 (2026-05-05 13:25)
- **타이틀 화면** 완성: 3D 사막 풍경 + 피라미드 + 트랙 + 주사위 낙하 애니메이션
- **게임 플레이** 미구현: "게임 시작" 버튼 클릭 시 아직 게임 화면 없음 (alert만)
- **진행률**: 약 30%

## 기술 스택
- **Three.js r128** (CDN: `https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js`)
- **순수 JavaScript**, 단일 HTML 파일 (현재 ~704줄)
- **WebGL**, PCFSoftShadowMap, ACESFilmicToneMapping
- **CSS 3D 변환**으로 UI 버튼 3D 효과
- **카메라**: PerspectiveCamera(40), 드래그로 시점 회전

## 3D 씬 구성
- **배경**: `0x0a0510` (짙은 남색) + FogExp2
- **지형**: 80×80 PlaneGeometry, sin/cos 파도로 모래언덕 구현, 중앙 평탄화
- **프로시저럴 텍스처**: Canvas 2D로 생성한 노이즈 모래 질감
- **조명**: Ambient + Directional(달빛, AACCFF) + Rim(6644AA) + Heat(FF8844)
- **달**: 구체 + 크레이터 + 3중 글로우 링 (CCAA88, AdditiveBlending)
- **별**: 2500개, 구면 좌표계 분포, 색온도 랜덤
- **피라미드**: 5단 박스 적층 + 금색 캡스톤 (emissive)
- **야자수**: 굴곡진 원통 줄기 + ConeGeometry 잎 10개
- **트랙**: RingGeometry(4.5~5.5), 16개 마커 타일, START/FINISH 폴 + 깃발
- **먼지 파티클**: 400개 점, 사막 공중에 부유
- **주사위**: 5색 큐브 + 모서리 베벨 + 도트, 8초마다 하늘에서 낙하 애니메이션

## 게임 규칙 (목표)
1. 5마리 낙타 (파랑/초록/빨강/노랑/하양) — Three.js 3D 모델로 구현 예정
2. 원형 트랙 16칸
3. 5개 주사위 (1,1,2,2,3,3)
4. 매 라운드: 주사위 굴리기 → 해당 낙타 이동 → 스택 처리
5. 베팅: 1등/꼴찌 예측 (x5 ~ x1 배당)
6. 8라운드 진행, 최종 코인으로 순위 결정
